# Contributing

Thank you for your interest in contributing to this project! ❤️

This repository hosts a **living meta-analysis**, updated regularly, alongside an interactive **Shiny application** for exploring the results.

If you like the project but don't have time to contribute, that's okay too! Other ways to show your support:

- Star the repository
- Share it with colleagues or on social media
- Mention it in your own papers

Before contributing, please discuss the change you wish to make via issue, email, or any other method with the repository maintainers. This ensures efforts are coordinated and avoids duplicated work.

## Types of Contributions

There are two main ways to contribute to this project:

1. **Data contributions** — adding or updating study data in the meta-analysis
2. **Shiny app contributions** — improving the interactive application

## Contributing Data 

If you have identified a new eligible study or a correction to existing data, please [open an issue](https://github.com/feracotommaso/living_SEB_review/issues/new) or contact the maintainers directly to discuss.

## Contributing to the Shiny App

⚠️ **Work in progress.** Guidelines for contributing to the Shiny App are currently being developed. If you have a suggestion or a bug fix in mind, please [open an issue](https://github.com/feracotommaso/living_SEB_review/issues/new) to start the conversation.

## Reporting Issues

If you find a bug in the app or an error in the data, please [open an issue](https://github.com/feracotommaso/living_SEB_review/issues/new) and include:

- A clear description of the problem
- Steps to reproduce (for app bugs)
- The study reference and field in question (for data errors)
- Your R session info (`sessionInfo()`) if relevant
